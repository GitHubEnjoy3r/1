Задача 12. Напишите программу для проверки пароля. Программа состоит из 2 тактовых полей. При вводе текста в одно из полей вызывается функция для проверки пароля. Условие 1. Проверка на длину пароля. Если пароль меньше 8 символов, то программа пишет в консоль «Короткий пароль». Условие 2. Проверка на совпадение паролей. Если пароли не совпадают, то программа пишет в консоль «Пароли не совпали». Если по итогу проверки, оба условия оказались верны, то программа выводит сообщение «Пароль принят».
